# Changelog

## v1.0.0

This release introduces breaking changes regarding the customization of the template.

### Refactorings

- Set color theme via `theme` parameter instead of loading it from a `config.yaml` file
- Customize CV experiences (work, education) entries via a single `experience` call with a `label` parameter to specify the "subtitle" 

### Misc

- Update to Typst v0.14.0

## v0.1.0

Initial template version.
